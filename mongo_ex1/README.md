Optical Store Database (Cul d'Ampolla)
Overview
This project contains the database for Cul d'Ampolla, an optical store that manages customers, glasses, supplers, employees, and sales.
The goal is to model the information in a way that is efficient, scalable, and easy to query from a graphical user interface.

Database Model
The database model is implemented in MongoDB, using a mix of embedded documents and references to avoid unnecessary data duplication.
